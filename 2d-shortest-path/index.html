Algorithm Information

Bellman-Ford Algorithm: Finds shortest paths from a source node to all other nodes in a weighted graph, even with negative edge weights.

Time Complexity: O(VE) where V is vertices and E is edges

Space Complexity: O(V)

Key Feature: Can detect negative cycles in the graph
